# California Housing Price Prediction

## Project Overview
This project demonstrates the complete Machine Learning workflow using the California Housing Dataset.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Jupyter Notebook

## Workflow
1. Data Loading
2. Exploratory Data Analysis (EDA)
3. Train-Test Split
4. Linear Regression Model Training
5. Model Evaluation
6. Visualization
7. Model Saving

## Dataset
California Housing Dataset from Scikit-Learn

## Outcome
Successfully built and evaluated a Linear Regression model to predict house prices.
